地区

地區為一層級結構，grade表示目前的地區是為於第幾層(1-base)，完整的地區資訊是從第一級逐層串接，地區的完整路徑可透過 itemAreaPath 屬性取得。

以下面的json為例， 石景山区 為地區層級的第5層，表示 石景山区 上還有四個層級，所以完整的的地區為資訊為: 亚洲-中国-北京-石景山区-石景山区

{
     "uuid" : "928DE107-52E3-4016-9CE1-AFC0FB1532AE",
     "no" : "2180",
     "name" : "石景山区",
     "grade" : "5",
     "itemAreaPath" : [ "亚洲", "中国", "北京", "石景山区", "石景山区" ]
}
